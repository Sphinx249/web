Our Process

1. Planning

Understang the client's requirements, goals, purpose.

Putting a plan together based on the client's requirements.

Creating a site map for pages to be created and web flow navigations.

Understanding the available resources and deciding on what technologies will be used to create the porduct.

2. Design

From the outline worked out, wireframe is created.

Working in team with designers to create a mock-up.

Tweaking the design untill the layout and the structure are finally worked out with all the ideas incorporated based on the requirements.

Have the writer create content related to client's business.

3. Development

Once prototype is approved, web development comes into play.

Successful design include good HTML/CSS code according to the web standards and functional code and database connections.

The content is added to ensure correct flow.

Changes needed to be done are made in this phase

4. Testing

Check for the complete functionality of all the links, page navigations, sunmissions and all functionalities implemented with in accordance to the requirements finalized.

Make sure no functionalities are missed out.

Perform cross-browser testing to ensure it works correctly in all browsers.

5. Delivery and Maintenence

After testing and development is done, transferring files to the live server and do a last minute run through and ensure its correctness.

Making changes to websites like frequent re-designs and keeping it up-to-date.
